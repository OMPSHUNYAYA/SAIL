RELEASE PURPOSE
==================================================

This document defines the final verification criteria for the SAIL v5.9.0 public reference release.

- the core runtime is synchronized
- evidence classifications are accurate
- synthetic structural conformance executions pass

- documented counts match

- deterministic structural invariants are preserved
- operational metadata is separated from structural certificates
- lifecycle guards fail closed
- accepted transitions are idempotent
- replay behavior remains deterministic
- federation invariants remain valid
- realization invariants remain valid
- pack states reflect actual structural resolution
- documentation and verification artifacts are synchronized

Release verification focuses on observable release behavior and evidence integrity rather than internal implementation preference.
